1. What is the different between ID and CSS

    ID meant to be unique within document, element can only have one id attribute. Class can be reused in multiple elements in the document, mainly for styling and targeting elements.

  2. What's the different between "restting" and "normalizing"

    "Restting" is meant to strip all default browser styling on element, e.g. margin, padding, font-size. "Normalizing" preserves usefaul default styles rather than "unstyling" everything. It also corrects bugs for common browser dependencies.

  3. CSS Box model

    content + border + padding + margin

  4. what is line-height?

    line-height is the height of a single line of document.

  5. how to make chrome support font size smaller than 12px?
